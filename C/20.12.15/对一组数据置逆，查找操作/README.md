1.void input(int a[],int n) 实现向数组a输入n个数；

3.void reserve(int a[],int n) 实现将数组a的前n个数置逆；  
4.int find(int a[],int n,int x) 实现在数组a的前n个数中查找x；若找到返回元素下标，若没找到，返回-1.
